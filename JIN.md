## JNI的学习

-------


### 什么是JNI
	
    java native interface(java本地开发接口)。相当于桥梁的左右，也可称为一种协议。是通过jni让java调用c语言或者c++代码，并且可以让c代表调java代码

-------

### 什么是NDK

    ndk: native develop kits(本地开发工具包).
